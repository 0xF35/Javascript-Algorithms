### Sha-1
``` javascript
console.log(Sha1.hash('hash this string'));
// 91aebc21fa63504351c19cc2add5e04478664d94
```

### Sha-256
``` javascript
console.log(Sha256.hash('hash this string'));
// eedb752a1e7a2691ebd896ce86d868c5ddc795419be5925030cbee768153700b
```
